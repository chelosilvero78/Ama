===
Ama
===

Ama es una libreria de utilidades hecha en Python para procesar datos de radar previamente
procesados por la librería `wradlib <http://wradlib.org>`_. Un típico uso de la librería
sería algo como esto::

    $ python -m ama

Ayuda
=====
Para listar la ayuda de Ama, utilizar el siguiente comando::

    $ python -m ama --help
